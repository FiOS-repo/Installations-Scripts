## Installation
1. Clone the repository:
```bash
git clone https://github.com/FiOS-Repo/easymultitool.git
```

2. Install `Easy Multi Tool`:
```bash
chmod +x ./install.sh && sudo bash ./install.sh
```

## Usage
To run the project, use the following command:
```bash
emt
```
